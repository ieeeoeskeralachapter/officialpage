# How to Run the IEEE OES Kerala Website

## Option 1: Direct Open
1. Navigate to the folder `c:\Users\harik\Desktop\webdev\ieee-oes-kerala\`.
2. Double-click the `index.html` file.
3. It should open in your default web browser.

## Option 2: VS Code Live Server
1. Open the project in VS Code.
2. Right-click on `index.html`.
3. Select "Open with Live Server".

## Troubleshooting
- If you see a blank page, try refreshing the browser (Ctrl + R or F5).
- Ensure `styles.css` and `script.js` are in the same folder as `index.html`.
